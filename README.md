# YanisManager

## Présentation

YanisManager est un système de gestion des données étudiantes en C++ basé sur la programmation orientée objet.  
Le programme permet de gérer des utilisateurs avec différents rôles : **ADMIN**, **PROF**, **STUDENT**, et de gérer les notes des étudiants.  

Fonctionnalités principales :  

- **ADMIN** : voir, ajouter, supprimer des étudiants  
- **PROF** : voir, ajouter, modifier, supprimer des notes  
- **STUDENT** : voir ses propres notes  
- Les menus sont en français et l’utilisateur peut se déconnecter pour revenir au login  

---

## Technologies utilisées

- Langage : C++17  
- Structures de données : `vector` (STL)  
- Programmation orientée objet  
- Fichiers séparés `.h` et `.cpp`
